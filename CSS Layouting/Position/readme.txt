<< POSITION RELATIVE >>
- Element akan naik ke dimensi berbeda tetapi ruang sebelumnya tetap ada, dan bergerak relatif

<< POSITION ABSOLUTE >>
- Element akan naik ke dimensi berbeda tetapi ruang sebelumnya sudah tidak ada dan Element di bawahnya akan mengisi ruang sebelumnya.
- Dan position absolute bergerak menyesuaikan Parentnya, jadi jika ingin memposisikan absolute maka akan bergerak sesuai parentnya.

<< POSITION FIXED >>
- Digunakan untuk mem-fix an Element pada posisi tertentu, (biasa digunakan untuk membuat topbar pada halaman website.z)
